# ☕ Cafe Management System

A simple console-based Cafe Management System developed using Python. The application allows users to view the menu, place orders, calculate bills automatically, and receive discounts on eligible orders.

## 🚀 Features

- Display cafe menu
- Place orders for multiple items
- Automatic bill calculation
- 10% discount on orders above ₹300
- User-friendly console interface

## 🛠️ Technologies Used

- Python

## 📂 Project Structure

```text
main.py
README.md
```

## ▶️ How to Run

1. Make sure Python is installed on your system.
2. Open the project folder in a terminal.
3. Run the following command:

```bash
python main.py
```
