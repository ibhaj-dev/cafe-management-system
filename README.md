# ☕ Cafe Management System

A simple Python-based Cafe Management System that allows customers to view the menu, place orders, and generate bills.

## 🚀 Features

- Display cafe menu
- Order multiple items
- Automatic bill calculation
- 10% discount on orders above ₹300

## 🛠️ Technology Used

- Python

## ▶️ How to Run

```bash
python main.py
