# ☕ Cafe Management System

A simple Python-based Cafe Management System that allows customers to view the menu, place orders, and generate a bill with discounts.

## 🚀 Features

- Display cafe menu with item prices
- Order multiple items with quantity selection
- Automatic bill calculation
- 10% discount on orders above ₹300
- User-friendly terminal interface

## 🛠️ Technologies Used

- Python 3

## 📂 Project Structure

```text
Cafe-Management-System/
│
├── cafe.py
└── README.md
```

## ▶️ How to Run

Open terminal and execute:

```bash
python cafe.py
```
